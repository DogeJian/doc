## 工具函数

> <b>获取字典</b>

```js
    getDictionary (data, key, value) {
      const result = {}
      data.forEach(v => {
        const realykey = v[key]
        result[realykey] = v[value]
      })
      return result
    }
```

> <b>groupBy</b>

```js
  groupBy (data, params) {
      const groups = {}
      data.forEach(v => {
        const group = JSON.stringify(v[params])
        groups[group] = groups[group] || []
        groups[group].push(v)
      })
      return groups
    }
```
